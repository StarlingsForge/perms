# Perms
This is a project to convert -rwxrw---- into easy human readable text.
## How to run
First clone the repository.
'''git clone [URL]'''
Once cloned the file needs permissions to allow it to run.
'''chmod u+x perms.sh'''
Then to run while in the directory.
'''./perms.sh [FileName]'''
